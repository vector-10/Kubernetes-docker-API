Steps for deployment to Azure App Service

1. Create an Azure Account - Done
2. Create a Resource Group - Done
3. Create an App Service plan - Done
4. Create app service/web app - Done
5. Create Github Repository - Done
6. Connect Github to Azure
7. Write script for production in server file
8. change base url to Azure url
9. Build Client files based on such URls
10. Push code to Github
11. Set up .env variables in Azure
